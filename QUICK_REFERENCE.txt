# ⚡ QUICK REFERENCE CARD

## 🚀 ONE COMMAND TO RUN EVERYTHING

### Windows
```
./start-docker.bat
```

### macOS/Linux
```
./start-docker.sh
```

### Manual (any OS)
```
docker-compose up --build
```

**Then open:** http://localhost:5000

---

## 📦 SINGLE ENVIRONMENT FILE: `.env.docker`

```env
NODE_ENV=development
PORT=5000
MONGO_URI=mongodb://mongo:27017/team-task-manager
JWT_SECRET=your-dev-secret-key-change-this-in-production-min-24-chars
JWT_EXPIRES_IN=7d
FRONTEND_URL=http://localhost:5000
RATE_LIMIT_WINDOW_MS=900000
RATE_LIMIT_MAX=200
```

✅ That's ALL the environment variables you need!

## 🛑 TO STOP

```bash
# Stop everything (keep database)
docker-compose down

# Stop everything (delete database)
docker-compose down -v
```

## 🐛 TROUBLESHOOTING

| Problem | Solution |
|---------|----------|
| Docker not found | Install Docker Desktop |
| Port 5000 in use | Edit docker-compose.yml: change 5000:5000 to 3000:5000 |
| Database error | Check .env.docker has MONGO_URI |
| Frontend not loading | Run: `docker-compose logs app` |
| Want to rebuild | Run: `docker-compose build --no-cache` |
